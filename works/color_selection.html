COLOR SELECTION

非デザイナーのための配色用のパレット。自身のカラーデザインを支援するために制作した。

Ver 3

統一性のある色選びを促すためにHSB(HSV)色空間で色を設定できるようにしている。 また色相(Hue)の選択を12段階に制限することで、色選びの複雑性を軽減させた。
制作したカラーセットは保存し、全世界のユーザーと共有することができる。

Ver 3での追加機能
Webアプリケーションとして制作した。＜リンクはこちら＞
色パターンをデータベースに保存できるようにした。 またこれによって様々なユーザーのカラーデザインを共有できるようになった。

Programing: JavaScript / PHP

Ver 1

デスクトップアプリケーションとして制作した。
利用頻度の高い色をあらかじめパレットとして用意した。 また、統一性のある色選びを促すためにHSB(HSV)色空間で色を設定できるようにしている。

Programing: Java

Ver 2

Webアプリケーションとして制作した。 カラーデザインをする機会は多岐にわたり、各デバイスに対応したアプリケーションを開発するよりもWebコンテンツとして展開したほうがより利便性が高まると考えWebアプリケーションとしての開発を開始した。

リメイクにあたり、Ver1から得られた反省を反映させたため、大幅に変わった点がいくつかある。

  • Ver1では色相(Hue)が細かく設定できる仕様であったが、複雑性が増し色選びが困難になると考え12段階で選択するようにした。
  • また、利用頻度の高い色をあらかじめ用意する方針は、多くのユーザーが使う上では無意味な制限を与えるため廃止した。

Programing: JavaScript
