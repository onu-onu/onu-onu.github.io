情報メディア特別演習 / 体重管理IoT

筑波大学 情報メディア創成学類の講義 情報メディア特別演習にて製作した。

既存の体重計をより生活に溶け込んだ形式にアップデートすることで，健康問題を日常的に解決するシステムのプロトタイプである。

Arduinoを用いた組み込み系デバイスの製作と，センサーを用いたフィジカルコンピューティングを学習する目的で製作した。 単なるIoT体重計にとどまらず，導電布を用いた薄型の体重測定手法の検討やBluetooth Low EnergyのBeacon，LINE message APIを利用したシステム構築など様々なスキルを養うことができた。

Programing: C++，PHP，java script
Hardware: ESP32
Other: LINE message API，LINE Simple Beacon，BluetoothLowEnergy
